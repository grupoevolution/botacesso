Webhook PerfectPay

Configure este URL na PerfectPay como webhook de pagamento aprovado.

Como identificar o lead
Coloque o Telegram ID do lead no campo utm_content do link de pagamento:

https://pay.perfectpay.com.br/...?utm_content=TELEGRAM_ID&utm_source=p1m

utm_source: p1m = mensal   p3m = trimestral   p12m = anual
