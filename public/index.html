Webhook PerfectPay
Configure este URL no painel da PerfectPay como webhook de pagamento aprovado:
Para identificar o lead, envie o Telegram ID do lead no campo utm_content do link de pagamento.
Exemplo: https://pay.perfectpay.com.br/...?utm_content=TELEGRAM_ID&utm_source=p1m

utm_source: p1m = mensal, p3m = trimestral, p12m = anual
